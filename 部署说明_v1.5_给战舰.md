# AI-Pulse v1.5 部署说明

## 修复内容
- **修复定时更新默认关闭问题**：确保定时更新默认开启，间隔更新默认关闭
- 修改文件：`api_server.py`、`frontend/src/views/Update.vue`

## 部署包
- 文件名：`ai-pulse-v1.5-scheduled-fix.tar.gz`
- 大小：558KB

## 部署步骤

### 1. 上传部署包到服务器
```bash
scp ai-pulse-v1.5-scheduled-fix.tar.gz admin@8.130.148.166:/tmp/
```

### 2. SSH登录服务器并执行部署
```bash
ssh admin@8.130.148.166
```

### 3. 在服务器上执行以下命令

```bash
# 停止当前服务
ps aux | grep api_server | grep -v grep | awk '{print $2}' | xargs -r kill
sleep 2

# 备份当前版本
cd /opt/behavior-recorder/ai-pulse
BACKUP_DIR="../ai-pulse-backup-$(date +%Y%m%d-%H%M%S)"
cp -r . "$BACKUP_DIR"
echo "备份完成：$BACKUP_DIR"

# 解压新版本
cd /opt/behavior-recorder/ai-pulse
rm -rf config src frontend *.py requirements.txt 2>/dev/null || true
tar -xzf /tmp/ai-pulse-v1.5-scheduled-fix.tar.gz
echo "解压完成"

# 启动服务
nohup python3 api_server.py > api_server.log 2>&1 &
sleep 3

# 验证服务
curl -s http://localhost:8887/health
ps aux | grep api_server | grep -v grep
```

### 4. 验证部署成功

访问前端页面：`http://8.130.148.166:8887`

进入"内容更新"页面，检查：
- ✅ 定时更新：默认**开启**状态
- ✅ 间隔更新：默认**关闭**状态
- ✅ 更新时间：8:00（默认）

### 5. 查看日志（可选）

```bash
tail -f /opt/behavior-recorder/ai-pulse/logs/fetch.log
```

## 注意事项
- 部署前已自动备份旧版本到 `../ai-pulse-backup-YYYYMMDD-HHMMSS`
- 如部署有问题，可回滚到备份版本
- 数据库配置保持不变，不会丢失已有数据
