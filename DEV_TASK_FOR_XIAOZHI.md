# 给小治的知识炼金术Agent V1.0开发指令
## 👋 核心要求：极简、快速验证，2天跑通全链路，只做必须做的，不做任何多余功能
---
## 🎯 V1.0开发优先级（严格按顺序做，非标注的功能全部延后）
### 🔹 第一步：爬虫模块开发（最高优先级，半天完成）
- 技术选型：直接用`trafilatura`库做通用HTML正文提取，不需要自己写爬虫逻辑，不需要适配公众号/知乎等反爬平台
- 功能要求：
  - 单URL抓取：`python main.py crawl --url https://xxx.com`，输出干净的Markdown正文+提取到的元数据（标题/发布时间/作者/来源）
  - 自动去噪：过滤掉广告/导航/推荐/评论区等无关内容
- 提交要求：代码提交+测试3个普通网页的抓取结果截图
### 🔹 第二步：LLM封装+核心处理逻辑开发（1天完成）
- LLM封装：复用AI视频项目的DeepSeek调用代码，支持按模型选择调用（V3.2/V4 Pro）
- 分类逻辑：
  - 时间维度：直接用代码根据发布时间判断，不需要LLM
  - 内容/场景维度：调用DeepSeek V3.2，按固定可选值分类，输出JSON
- 评分逻辑：调用DeepSeek V4 Pro，输出重要性/稀缺性/实用性三个维度的得分（0-10分）
- 核心提炼：调用DeepSeek V4 Pro，输出3-5个核心观点+视频适用场景
- 综合评分：代码实现最终分计算+星级映射逻辑
- 提交要求：单步测试通过，每个环节能单独运行输出正确结果
### 🔹 第三步：全链路打通+存储归档（半天完成）
- 流程串联：输入URL后自动跑完「抓取→分类→评分→提炼→计算最终分」全流程
- 存储：
  - 结构化数据全部存入SQLite，字段包含所有元数据、分类、评分、核心观点
  - 原始Markdown文件按`内容维度/时间维度`的目录结构自动保存到本地
- 命令行优化：单条命令`python main.py process --url https://xxx.com`就能跑完所有流程，输出最终结果
- 提交要求：全链路跑通测试，至少5个URL测试成功，归档正确
