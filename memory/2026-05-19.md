## 📌 部署建议
### Nginx配置（运维层）
```nginx
# 限流配置
limit_req_zone $binary_remote_addr zone=api:10m rate=600r/m;
# UA过滤
if ($http_user_agent ~* "^(curl|wget|python-requests)/*") {
    return 403;
}
# 接口代理
location /api/public/ {
    limit_req zone=api burst=40 nodelay;
    proxy_pass http://localhost:7861;
}
```
